# macbook

My NixOS configurations for my macbook. This requires the use of the `flake.modules.darwin` aspect,
as well as the `flake.modules.homeManager` aspect.

Based on the example [here](https://github.com/Doc-Steve/dendritic-design-with-flake-parts/tree/69edacdb5a4a6ca71d649bb8eb62cf8c630c8627/modules/hosts/macbook%20%5BD%5D).
